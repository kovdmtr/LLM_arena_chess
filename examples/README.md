# Примеры вывода

## `sample-game/`

Готовый пример одной партии — то, что арена выдаёт на выходе. Сгенерирован
детерминированно, без сети и без API-ключей (скриптованные ходы), но с **настоящим
пост-анализом Stockfish**, скриптом `scripts/generate_sample_game.py`.

Партия — детский мат (Scholar's mate), `GPT-4o` (белые) против `Claude Opus 4.8`
(чёрные), `1-0` (мат на 4-м ходу). Белым на первом ходу выдаётся подсказка движка
(★, D-010). Пост-анализ (★, D-009) размечает ходы: чёрные получают `inaccuracy` и
решающий `blunder` (ход `Nf6`, упустивший угрозу мата на f7) — он попадает в
ключевые моменты.

| Файл | Что это |
|------|---------|
| `game.json` | Источник истины: полный лог партии (ходы, рассуждения, подсказка, оценки/классификация, сводка анализа). Без секретов (D-003). |
| `game.pgn`  | PGN для lichess/chess.com; рассуждения — комментариями `{ … }`. |
| `report.html` | Self-contained отчёт: одна доска + интерактивный плеер (перемотка ходов), бейджи классификации, сводка точности, ключевые моменты. Открывается в браузере, без внешних файлов. |

Перегенерировать: `python scripts/generate_sample_game.py`. Если Stockfish
недоступен, артефакты всё равно создаются — но без ★ подсказок/анализа (база, D-008).
